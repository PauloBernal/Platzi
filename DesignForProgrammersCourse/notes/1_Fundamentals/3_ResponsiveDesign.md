# Responsive Design

Es una metodología de diseño que permite tener un diseño adaptable a distintos tipos de pantalla. Algunos parámetros para realizar este tipo de diseño son:

- Empezar por dispositivos móviles (Responsive Design)
- Separar las capas de contenido y funcionalidad
- Utilizar sistemas de grillas, columnas y filas

## Metodologías para hacer Responsive Design

### Mejora progresiva (Progressive enhancement)

Consiste partir de una base sólida en la que se tiene los elementos esenciales y se va añadiendo capas de complejidad a medida que se avanza. Esto permite que usuarios con dispositivos de gama baja puedan acceder al contenido y que usuarios con mejores dispositivos o conexión disfruten de una experiencia más enriquecida.

## Degradación agraciada (Graceful degradation)

Es lo contrario a la anterior metodología y consiste en partir de un diseño completo e ir removiendo capas de complejidad para ir adaptando el diseño a otros dispositivos más pequeños.

## Metodologías aplicadas al código

Estas metodologías también se aplican al momento de realizar el código (las mismas metodologías de diseño). Por ejemplo en Progressive enhancement los equivalentes serían:

| Diseño | Código |
| ------ | ------ |
| 1. Contenido | 1. Data |
| 2. Wireframes | 2. HTML |
| 3. Diseño Visual | 3. CSS |
| 4. Animaciones | 4. JavaScript |
