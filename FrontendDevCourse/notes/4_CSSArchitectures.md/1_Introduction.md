# ¿Qué son y para que sirven las arquitecturas CSS?

Una arquitectura de código es un sistema de organización de elementos para manejar proyectos grandes. Los objetivos de una arquitectura CSS son:

- Debe ser predecible
- Debe ser reutilizable
- Debe ser mantenible
- Debe ser escalable

## Buenas prácticas

- Establecer reglas con el equipo
- Explicar la estructura base
- Establecer estándares de codificación
- Evitar largas hojas de estilo
- Documentación
