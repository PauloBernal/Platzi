# Anatomía de una declaración CSS

Los estilos en CSS se componen de múltiples declaraciones depediendo de los estilos de cada elemento o cada grupo de elementos. Estas declaraciones se componen de:

- Selector.- Hace referencia a uno o más elementos HTML
- Declaración.- Es la unión de una propiedad con su respectivo valor.
- Propiedad.- Es un aspecto a cambiar en el elemento ya sea color de fuente, tamaño, margen, color de fondo, etc.
- Valor.- Es el valor que se la da a la propiedad para modificar el aspecto.

~~~css
selector {
  property1: value1;
  property2: value2;
  ...
  propertyN: valueN;
}
~~~
